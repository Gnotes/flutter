# 初始化项目

```bash
flutter create my_app
```

## IOS

- 启动模拟器

    ```bash
    open -a Simulator
    ```
- 启动项目

    ```bash
    # cd my_app
    flutter run
    ```

## Android

- 启动模拟器

    `Tools` -> `AVD Manager`  
    ①创建模拟器  
    ②启动模拟器  

- 启动项目

    ```bash
    # cd my_app
    flutter run
    ```

## VSCode中启动

在 flutter 项目中， `Cmd + Shift + P` 可打开扩展面板，搜索 Flutrer: Launch Eimulator 可以启动、创建模拟器
